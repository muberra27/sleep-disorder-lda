# Sleep Disorder Classification using LDA

This project applies Linear Discriminant Analysis (LDA) to classify sleep disorders
using lifestyle, physiological, and demographic features.

## Dataset
Sleep Health and Lifestyle Dataset containing variables such as:
- Age
- Gender
- Occupation
- Sleep Duration
- Stress Level
- Blood Pressure
- Heart Rate
- BMI Category

## Methods
- Data cleaning and preprocessing
- Feature scaling of numerical variables
- Full and reduced LDA models
- Model evaluation using repeated train/test splits
- Confusion matrices and average test error

## Technologies
- R
- MASS
- tidyr

## How to Run
1. Clone or download the repository
2. Open `lda_sleep_disorder.R` in RStudio
3. Run the script
